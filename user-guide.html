CADesk User Guide

1. Introduction

CADesk is a web-based document management portal developed for Chartered Accountant offices. This system allows Admin, Employee, and Client to securely manage, upload, and access client documents online. The system ensures safe storage, easy access, and role-based login.

2. System Roles

  • Admin – Can add clients, upload documents, and manage users.
  • Employee – Can upload and manage client documents.
  • Client – Can view and download their documents.

3. Login Instructions

  1. Open the CADesk website.
  2. Select your role (Admin / Client / Employee).
  3. Enter your registered email and password.
  4. Click on Login button.
  5. If details are correct, dashboard will open.

4. Admin Guide

  • Admin can login using admin credentials.
  • Admin can create new client accounts.
  • Admin can upload documents for clients.
  • Admin can view all clients list.
  • Admin can manage files and records.

5. Employee Guide

  • Employee can login using employee credentials.
  • Employee can upload client documents.
  • Employee can view client list.
  • Employee cannot create admin accounts.

6. Client Guide

  • Client logs in using provided email and password.
  • Client dashboard shows uploaded documents.
  • Client can download files.
  • Client cannot upload or edit files.

7. Uploading Documents

Admin or Employee selects client → chooses file → uploads document. The document is stored securely in the system and visible to that client only.

8. Downloading Documents

Client logs in → opens dashboard → selects file → clicks download button.

9. Logout

User should click Logout button after completing work to keep the account secure.

10. Error Messages

  • Invalid email format – Enter correct email.
  • Invalid email or password – Check login details.
  • User not found – Contact admin.

11. Security Features

  • Role-based login system.
  • Secure authentication using Firebase.
  • Each client can see only their files.
  • Admin controls all users.

12. Conclusion

CADesk provides a simple, secure, and efficient way to manage client documents for Chartered Accountant offices. The system reduces manual work and allows clients to access documents anytime.
